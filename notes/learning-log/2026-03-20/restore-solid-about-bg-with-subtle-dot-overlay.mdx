## Context
The About section needed a subtle texture treatment without reducing text contrast.

## Goal
Keep a solid dark section background and place a low-opacity texture overlay above it, while preserving content readability.

## Files Changed
- `components/about-section.tsx`

## Implementation Notes (key logic explained)
- Replaced gradient section background with solid `bg-[#0a0a0f]`.
- Kept/used an absolutely positioned overlay layer:
  - `absolute inset-0`
  - `pointer-events-none`
  - `opacity-[0.16]`
  - radial dot pattern via `backgroundImage`
  - `backgroundSize: '18px 18px'`
- Content stays in a separate `relative` wrapper with higher stacking (`z-[3]`) so text is not dimmed by the overlay.

## Key Decisions (and alternatives considered)
- Decision: solid base + lightweight dot overlay.
- Alternative: full-section gradient background. Rejected to match requested visual direction and stronger text consistency.

## Assumptions
- Dot texture should be subtle and non-interactive.

## Failure Modes
- If overlay opacity is increased too far, readability may degrade.
- If content z-index is lowered, texture can visually interfere with text.

## How to Test (commands + steps)
1. Run `npm run dev`.
2. Open `/terrain` and scroll to About.
3. Confirm background is solid dark with subtle dot texture.
4. Confirm text remains crisp and high-contrast.

## Next Steps
- Tune overlay intensity by adjusting `opacity-[0.16]` or dot color alpha.
